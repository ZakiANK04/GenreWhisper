====================================================================================================
PROJECT IDENTITY
====================================================================================================

Project name:
GenreWhisper

Core tagline:
"Unveiling Hidden Genres from the Whispers of Readers"

Research question:
"Can reader reviews alone predict a book's genre/category? And do review sentiments reveal hidden genre biases?"

Core concept:
The project treats reader reviews as signals of a book's hidden genre identity. Instead of trusting official metadata alone, it asks whether review language itself can reveal the true semantic category of a book. It then extends the analysis to ask whether some genres attract systematically different sentiment patterns, review lengths, or emotional intensity.

Project components:
1. A notebook-based NLP pipeline.
2. A cinematic Next.js website that presents the project in an immersive, vintage-library aesthetic.

This is not just a machine learning assignment. It is a fusion of:
- literary interpretation,
- practical NLP,
- model evaluation,
- product storytelling,
- visual world-building.

====================================================================================================
DATASET AND TECHNICAL FACTS YOU MUST PRESERVE
====================================================================================================

Dataset:
Amazon Books Reviews dataset from Kaggle

Files:
- books_data.csv
- Books_rating.csv

Exact columns used:
From books_data.csv:
- Title
- description
- authors
- categories

From Books_rating.csv:
- Title
- review/text
- review/summary
- review/score

Join logic:
- The project merges the two files on normalized Title.
- It extracts a primary category from the categories field.

Preprocessing:
- spaCy
- tokenization
- lemmatization
- stop-word removal
- punctuation removal

Embedding model:
- custom FastText model
- 100-dimensional
- trained on the cleaned review corpus

Classification model:
- Logistic Regression
- class_weight = "balanced"

Why class weights matter:
- genre classes are imbalanced,
- some categories dominate review volume,
- macro metrics matter because minority classes should not disappear inside a weighted average.

Evaluation used:
- precision macro
- recall macro
- F1 macro
- precision weighted
- recall weighted
- F1 weighted
- confusion matrix

Deployment angle:
- model artifacts exported as ONNX
- intended for integration into a Next.js website
